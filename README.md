# timeplex
# This is a project for IKEA Uppsala

Todo tutorial: https://www.digitalocean.com/community/tutorials/build-a-to-do-application-using-django-and-react

To start backend:
- Go to timeplex folder
- pipenv shell 
- cd backend
- python manage.py runserver

To start frontend:
- cd frontend 
- npm start
